# 53. Maximum Subarray
**Difficulty:** Easy

## URL

https://leetcode.com/problems/maximum-subarray/

## Solution

### Approach 1: Greedy

$O(n)$, one pass.

The code is shown below:

```c++
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        max_subsum = float('-inf')
        cur_sum = 0
        for i in range(len(nums)):
            cur_sum += nums[i]
            if cur_sum > max_subsum:
                max_subsum = cur_sum
            if cur_sum < 0:
                cur_sum = 0
        return max_subsum
```
